## Project Description

The Banking System Console is a C++ console-based application that simulates basic banking operations. It allows users to create an account, deposit money, withdraw money, check account balance, and view account details through a menu-driven interface.

## Features

- Create New Account
- Deposit Money
- Withdraw Money
- Check Balance
- View Account Details
- Simple Menu-Driven Interface
- Input Validation

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)

## How to Run

1. Open the project in any C++ IDE (CodeBlocks, Dev-C++, Visual Studio Code, etc.).
2. Compile the `banking_system.cpp` file.
3. Run the program.
4. Select options from the menu to perform banking operations.
